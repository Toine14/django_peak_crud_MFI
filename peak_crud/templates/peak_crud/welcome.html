Welcome to peak API

To use this API please go there

To get all peak registered in the database go to domain_name/api/peak

You can use the form or POST to url "domain_name/api/peak" in order to create a new peak

The POST body request should be like below:

{ "name":"Mont_Blanc", "elevation":4808, "location": { "type": "Point", "coordinates": [ -4.078734191134572, 23.78021472413093 ] } }

Once created you can UPDATE, DELETE by going to /api/peak/peak_id
